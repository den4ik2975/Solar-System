3. Эллипсы с поворотом (красиво выглядит)
   Добавляешь угол поворота самого эллипса:

x_temp = a * cos(angle)
y_temp = b * sin(angle)

x = centerX + x_temp * cos(rotation) - y_temp * sin(rotation)
y = centerY + x_temp * sin(rotation) + y_temp * cos(rotation)


Для Луны вокруг Земли — та же формула, только центр не Солнце, а текущая позиция Земли.
